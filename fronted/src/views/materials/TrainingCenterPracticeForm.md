# 专项实战表单设计

## 新增字段说明

### 1. 场景信息（更明确的定位）
- **场景类型** (scenarioType): 下拉选择 - 考试模拟/项目实战/竞赛训练
- **场景详情** (scenarioDetail): 具体说明，如"软考中级"、"蓝桥杯Java组"、"电商项目"

### 2. 背景与约束（定制化核心）
- **背景说明** (background): 考试大纲、项目需求文档、竞赛规则等
- **指定技术栈** (techStack): JSON数组，明确要求使用的技术
  ```json
  ["Spring Boot 2.x", "Vue 3", "MySQL 8.0"]
  ```
- **限制条件** (constraints): JSON对象，各种约束
  ```json
  {
    "timeConstraints": ["3小时完成", "不得查阅资料"],
    "techConstraints": ["必须使用Maven", "JDK 11"],
    "resourceConstraints": ["单机环境", "内存限制2G"]
  }
  ```

### 3. 参考与兼容（帮助实施）
- **参考资料** (referenceMaterials): JSON数组，提供学习资料
  ```json
  [
    {"title": "官方文档", "url": "https://..."},
    {"title": "历年真题", "url": "https://..."}
  ]
  ```
- **模板兼容性说明** (compatibilityNotes): 文本说明
  - 需要注意的版本兼容问题
  - 需要额外配置的环境
  - 可能遇到的坑和解决方案

## 表单结构

1. **场景定位** - 明确是什么类型的专项训练
2. **背景说明** - 为什么要这样训练
3. **技术约束** - 必须用什么技术
4. **限制条件** - 时间、资源等约束
5. **实战内容** - 具体做什么
6. **评估标准** - 怎么算完成
7. **参考资料** - 哪里可以学习
8. **兼容说明** - 需要注意什么

这样的设计能够支持各种定制化场景！
